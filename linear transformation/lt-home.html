What is a Linear Transformation?

A linear transformation is a function between vector spaces that preserves two key operations:

  • Addition: T(u + v) = T(u) + T(v)
  • Scalar multiplication: T(c × u) = c × T(u)

In simpler terms, a linear transformation takes vectors and transforms them in a way that keeps their structure consistent. This means adding vectors before or after transforming gives the same result, and multiplying a vector by a number before or after transforming it also gives the same result.

How Do Linear Transformations Work?

Linear transformations are often represented by matrices. When you multiply a matrix by a vector, the result is a new vector that has been transformed according to specific rules.

For example, applying a transformation matrix to a vector can scale, rotate, reflect, or shear the vector. A transformation matrix like:

A = [ [2, 0],
[0, 3] ]

will stretch a vector horizontally by 2 and vertically by 3. These operations are essential for modeling and simulating real-world behavior in two or more dimensions.
